# CI/CD Калькулятор Проект

Это простой проект для демонстрации CI/CD (Continuous Integration/Continuous Deployment).

## Что это такое?

**CI (Continuous Integration)** - автоматическая проверка кода при каждом изменении
**CD (Continuous Deployment)** - автоматическое развертывание если всё хорошо

## Как это работает:

1. Вы пишете код и пушите в GitHub
2. GitHub Actions автоматически запускает тесты
3. Если тесты пройдены - всё хорошо
4. Если тесты упали - вы получаете уведомление

## Структура проекта:

```
ci_cd_calculator_project/
├── calculator.py          # Основной код калькулятора
├── test_calculator.py     # Тесты для калькулятора
├── requirements.txt       # Зависимости Python
├── .github/workflows/     # Папка для CI/CD конфигурации
│   └── ci.yml            # Файл настройки GitHub Actions
└── README.md             # Этот файл
```

## Как запустить локально:

```bash
# Запуск калькулятора
python calculator.py

# Запуск тестов
python -m pytest test_calculator.py -v
# или
python -m unittest test_calculator.py -v
```

## Что делает CI/CD:

При каждом push в репозиторий GitHub Actions будет:
1. Создавать виртуальную машину Ubuntu
2. Устанавливать Python 3.9
3. Устанавливать зависимости
4. Запускать все тесты
5. Показывать результат

## Попробуйте сами:

1. Создайте репозиторий на GitHub
2. Загрузите туда этот код
3. Посмотрите в вкладку "Actions" - там будет работать робот!
4. Попробуйте изменить код и сломать тест - робот покажет ошибку
